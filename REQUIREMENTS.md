# 烟蒂股筛选器网站 - 需求文档

## 项目概述
一个轻量化的A股/港股烟蒂股筛选网站，用户可以自定义选股策略，也可以使用预设模板。

## 核心功能

### 1. 数据接入
- 接入A股、港股实时/延迟行情数据接口
- 支持字段：市盈率(PE)、市净率(PB)、股息率、市值、ROE、负债率等

### 2. 选股策略编辑器
- 可视化策略编辑界面
- 支持字段间的加减乘除运算
- 支持设置阈值条件（大于、小于、等于、区间）
- 支持多条件组合（与/或关系）
- 策略保存到本地浏览器（localStorage）

### 3. 预设烟蒂股模板（3-4个）
- **经典烟蒂股**：低PB + 低PE + 高股息
- **格雷厄姆式**：PE<15 + PB<1.5 + 负债率<50%
- **深水炸弹**：PB<0.8 + 市值<100亿 + 正收益
- **高息防守**：股息率>5% + 连续分红5年 + PE<20

### 4. 筛选结果展示
- 表格展示符合条件的股票
- 支持排序、导出CSV
- 显示关键指标卡片

### 5. UI/UX要求
- 现代化简洁设计，深色/浅色模式
- 响应式布局（PC+移动端）
- 加载动画、过渡效果

## 技术栈
- **前端**：HTML5 + Tailwind CSS + Vanilla JavaScript
- **数据接口**：AKShare（免费A股港股数据）或 腾讯财经API
- **部署**：静态网站，可放GitHub Pages/Vercel

## 接口方案
使用AKShare（Python库，可通过FastAPI封装成HTTP接口）
或直接使用腾讯财经API（前端可直接调用）
